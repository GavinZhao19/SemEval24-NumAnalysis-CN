# NumAnalysis-Chatglm3-6B

## Model Overview

- **Name**: NumAnalysis-Chatglm3-6B
- **Base Model**: ChatGLM3-6B
- **Training Regime**: Full-parameter Supervised Fine-Tuning (SFT)
- **Training Duration**: 6 epochs
- **Optimal Checkpoint**: Step 8000

## Model Files

1. `pytorch_model-00001-of-00002.bin`
